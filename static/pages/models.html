ავტომობილების მიმოხილვა

A class

A-კლასი გამოირჩევა ყველაზე ოპტიმალური აეროდინამიკური კოეფიციენტით. იმდენად დინამიურით, რომ მცირდება საწვავის ხარჯი და ძრავის ხმაური.

B class

ახალი B კლასი აქ არის. უფრო დინამიური, უფრო კომფორტული და უსაფრთხო ვიდრე ოდესმე. რადგან უსაფრთხოება ზოგჯერ ყველაფერია.

c class

ახალი C-კლასი აერთიანებს სპორტულობას და ფუფუნებას თანამედროვე ფორმაში. დიზაინი: სპორტული სტილის გამოხატულება.ახალ C-კლასში პროგრესულობა ერწყმის ფუფუნებას.

E class

მზად არის მარტივად აითვისოს ყველაზე დიდი გამოწვევაც კი. აღმოაჩინეთ ახალი E-Class სალონის შესანიშნავი თვისებებიკიდევ უფრო დინამიკური, ელეგანტური და მდიდრული, როგორც არასდროს..

G class

G-კლასის უნიკალური დნმ წარმოიშვა დაახლოებით 40 წლის წინ. მისი უნიკალური მახასიათებლები, რომლებიც დღემდე განაგრძობენ მისი შეუდარებელი გარეგნობის განსაზღვრას.

S class

ახალი S-Class ორიენტირებულია მთავარზე, უნიკალური კომფორტისა და უსაფრთხოების ტექნოლოგიები, რომელსაც შეგიძლიათ ენდოთ, იქნებით მძღოლი თუ უკანა მგზავრი.
